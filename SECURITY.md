# Security Policy

## Supported Versions

This project is pre-1.0 (solo-maintained). Only the **latest published release**
receives security fixes — there is no multi-version support matrix.

## Reporting a Vulnerability

Please report security issues privately via GitHub's
[**Security Advisories**](https://github.com/micschr0/gradle-test-shard-plugin/security/advisories/new)
("Report a vulnerability" in the Security tab), not as a public issue.

This is a solo-maintained project — there's no SLA, but reports are triaged as
soon as reasonably possible. A fix (or at least an acknowledgment) typically
follows within a few days.
